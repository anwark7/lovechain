The Problem

Komitmen cuma janji. Sampai putus.

Kenali Dinda — dua tahun LDR, sudah bahas nikah, bahkan patungan nabung. Pasangannya ghosting: hilang, dana patungan dibawa, tanpa penjelasan. Tidak ada satu pun mekanisme netral untuk menagih kejelasan — atau dananya.

How It Works

Kunci dana. Cairkan lewat aturan.

01
💬

Create.

User A susun kontrak: durasi, deposit, witness, aturan.

02
🤝

Accept.

User B setujui & deposit. Kontrak jadi aktif.

03
🔒

Lock.

Dana kedua pihak terkunci di smart contract.

04
🔏

Prove.

Ajukan outcome + bukti. Witness / partner konfirmasi.

05
✅

Release.

Threshold tercapai → payout otomatis sesuai hasil.

🛡️ Jaring pengaman. Partner hilang atau witness pasif? Dana tetap bisa ditarik lewat timeout — tidak pernah stuck.

Demo Scenarios · jalankan LIVE di app

Tiga jalur, end-to-end.

💍

Wedding Unlock

Buat kontrak → deposit → ajukan bukti nikah → partner konfirmasi → 3/5 witness approve → dana kembali + wedding badge.

🕊️

Peaceful Exit

Request breakup → partner approve → cooling period lewat tanpa dispute → deposit kembali 50:50.

⚠️

Breach Resolution

Ajukan klaim + challenge bond + bukti → partner challenge → 4/5 witness vote → dana pihak melanggar dialihkan.
